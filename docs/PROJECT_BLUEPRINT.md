# SCPRAS Project Blueprint

SCPRAS is the Smart Construction Progress and Resource Analysis System developed for construction monitoring and resource control in Rwanda.

## MVP Phase 1

- Authentication and user roles
- Worker management
- QR/RFID/NFC attendance
- Project management
- Material tracking
- Reports and analytics dashboard

## Phase 2

- AI analytics
- Workforce forecasting
- Smart recommendations
- Mobile app

## Phase 3

- Face recognition
- BIM and AutoCAD integrations
- Drone monitoring
- Predictive AI
